Как создать-бота в дискорде?


Для этого вы должны определиться, какого бота вы хотите создать фан-бота или модераторского бота.


Ютуб-канал автора:

Создание-бота

Что бы создать бота вы должны зайти в портал разроботчиков дискорда, потом заходите вкладку приложение (applications).

И создаете приложение (applications), но это ещё не всё.


Потом заходите вкладку bot и нажмите там кнопку add bot.

Поздравляю вы создали бота!



После того как вы создали бота, вы должны устонавить библиотеку дискорд пайтон, если вы используете программу visual studio code или pycharm то пишите в терминал.


pip install discord.py


Если вы используйте другую программу для кода, то открывайте командную строку пишите cd и ссылку на папку вы должны нажать сверху папки и вы получаете ссылку. Потом в командной строке установливаете библиотеку пайтон.


Кодирование

Создаем папку config.py (так будет удобнее писать код)


token = "Токен вашего бота"
prefix = "Префикс вашего бота"

После этого создаем папку bot.py и там пишем.

import discord
from discord.ext import commands
from config import settings

Вам надо ещё подключить префикс бота в bot.py

bot = commands.Bot(command_prefix = settings["prefix"])

Потом в конце кода вы должны написать

bot.run("token")

Что бы запустить бота пишите:

@bot.event
async def on_ready():
print(------)
print(Бот в онлайне)
print(------)

Что бы у бота был статус пишите:

await bot.change_presence(activity=discord.Game(name="дискорд-пайтон"))

И наконец запуск бота что бы бот был в онлайне пишите:

#файл где находится бот py bot.py
